Instruções complementares para instalação do Aelius no Linux Ubuntu

Instalação do Aelius e do HunPos

Baixe a nova versão do Aelius, abra o arquivo INSTALL.html e o leia com atenção.


No Terminal do Ubuntu, execute o comando


sudo apt-get synaptic


para instalar o gerenciador de instalação de programas Synaptic, caso ainda não tenha instalado na sua máquina.


Abra o Synaptic por meio do menu Iniciar do Ubuntu (ou análogo) ou por meio do Terminal com o comando:


sudo synaptic


No Synaptic, procure por python-idle e python-devel (ou python-dev), instalando esses programas caso não estejam instalados. Faça o mesmo com PyYAML, Numpy e Matplotlib.


Caso PyYAML e Matplotlib não estejam disponíveis no Synaptic, baixe os respectivos arquivos compactados .tgz. Eles deverão ser baixados para a pasta $HOME/Downloads (em que a variável de ambiente $HOME tem valor do tipo /home/francy etc.). No Terminal, execute:


cd Downloads (muda o diretório atual de trabalho para a pasta Downloads)1


ls (lista os arquivos do diretório atual de trabalho)


Descompacte o arquivo tgz:


tar -xvzf pyyaml (tecle TAB para o Terminal completar o nome do arquivo)


Realizada a descomparação, execute:


cd pyyaml (tecle TAB para o Terminal completar o nome do arquivo)


Dentro da pasta do pyyaml, execute:


sudo python setup.py install


Faça o mesmo para o Matplotlib se for o caso.


Faça o mesmo procedimento para o tgz da versão recomendada do NLTK.


Através do Terminal, entre no idle, executando:


idle


Teste se as bibliotecas estão instaladas:


>>> import numpy, matplotlib, yaml, nltk


Verifique a versão do NLTK:


>>> nltk.__version__


Baixe a versão do HunPos para o Linux, descompacte o arquivo e coloque os dois binários na pasta recomendada no arquivo INSTALL.html do Aelius.


Coloque as pastas Aelius e aelius_data nos locais recomendados, conforme arquivo INSTALL.html.


Feche o idle. No Terminal, execute, a partir do diretório $HOME (que é o diretório de trabalho do Terminal quando iniciado):


sudo gedit .profile


No final desse arquivo, inclua as seguintes linhas:


CLASSPATH="$HOME/jmx/mxpost.jar"

export CLASSPATH

PATH="${PATH}:$HOME/jmx"

export PATH


PATH="${PATH}:$HOME/Applications/bin"

export PATH


PYTHONPATH="${PYTHONPATH}:$HOME/Applications"

export PYTHONPATH


Salve o arquivo. Feche o gedit e o Terminal. Faça logoff da conta (finalize a sessão). Faça login. Crie a pasta analises no diretório $HOME. Coloque nessa pasta um texto no formato .txt para ser etiquetado (por exemplo teste.txt). Entre no Terminal. A partir do Terminal, entre na pasta analises:


cd analises


Verifique se o arquivo teste.txt está na pasta:


ls


A partir de $HOME/analises, via Terminal, entre no IDLE:


idle


Teste o Aelius com o AeliusHunPos:


>>> from Aelius import Extras, Toqueniza, AnotaCorpus

>>> h=Extras.carrega(“AeliusHunPos”)

>>> t=”teste.txt”

>>> AnotaCorpus.anota_texto(t,h,"hunpos")

Instalação do MXPOST

Na pasta $HOME, crie a pasta jmx e coloque dentro dela o conteúdo do arquivo tgz descompactado. Verifique no Terminal se os arquivos são os listados abaixo:


ls -m jmx/

LICENSE.html, MXPOST.html, MXTERMINATOR.html, eos.project, mxpost, mxpost.jar, mxterminator,

tagger.project, trainmxpost, trainmxterminator


Através do Synaptic, instale Java.
